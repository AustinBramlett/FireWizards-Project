# FireWizards
Description

This is a Java problem solving system where users can create their accounts, solve coding problems, and track their progress. The system allows students to view problems based on difficulty and type, submit solutions, and earn achievements as they improve.

## Features

Create and manage user accounts

Login and logout

View problems by difficulty and type

Submit solutions to problems

Track progress and achievements

Comment on problems

Submit reports

Admins can ban, mute, or remove users and content

## User Roles

Student – Solves problems and tracks progress

Contributor – Adds or removes problems

Administrator – Manages users, reports, and problems
